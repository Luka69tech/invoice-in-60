## Daily Workflow

### Every Day: 1 SEO Blog Post
Write 800–1,500 word blog posts targeting 1 keyword. Publish to `~/invoice-in-60/app/blog/` directory (or create markdown files that can be added to the site).

**Post Structure:**
```
Title: [Exact H1 targeting keyword]
Meta description: [150 chars, includes keyword + CTA]
Introduction: [Hook, problem, solution]
H2: [Supporting section with keyword variations]
H2: [Supporting section]
H2: [How-to or listicle section]
CTA: "Try Invoice In 60 free → [link]?utm_source=seo&utm_medium=blog
```

**Quality Standards:**
- Minimum 800 words
- Keyword in title, first 100 words, 1 H2
- Internal link to main site
- External link to 1–2 authoritative sources
- Unique, helpful, actionable content (not thin AI content)

### Weekly: Backlink Building
- Comment on 5–10 relevant blog posts with value-adding comments
- Include link to relevant blog post (not homepage)
- Target: accounting blogs, freelance blogs, small business blogs
- Track: comment URL, site, dofollow/nofollow

### Ongoing: On-Page Optimization
- Optimize existing pages for target keywords
- Add internal links from homepage to blog posts
- Check: title tags, meta descriptions, heading structure, image alt text
- Report any technical SEO issues to CEO
